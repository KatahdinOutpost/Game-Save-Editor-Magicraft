Game Save Editor for Magicraft

This tool will decode the save file with XOR encryption for JSON editing. Upon saving the changes, the file will be encoded with encryption again.

Magicraft save files are stored on Windows PCs at C:\Users\[YourName]\AppData\LocalLow\bolang\Magicraft.

Upload your WorldData file. Each one is related to one of 3 save slots. WorldData contains the character information.
